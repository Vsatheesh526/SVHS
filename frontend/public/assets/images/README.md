# Static Images (Public Assets)

Store images here that are referenced in HTML or CSS and don't need webpack/Vite processing.

Example:
- favicon.ico
- background images
- logos
- hero images

Usage in HTML:
```html
<img src="/assets/images/hero.jpg" alt="Hero" />
```

Usage in CSS:
```css
background-image: url('/assets/images/pattern.png');
```
